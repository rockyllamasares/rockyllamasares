## 🧠 How I Work

I don't rely on motivation — I rely on **systems**.

Every day runs on the same framework:

```
Morning  →  Pre-mortem: How will I fail today? Eliminate it.
Start    →  Pick only 3 priorities. No more.
First    →  Win small. Build momentum early.
Focus    →  Deep work = no phone, no notifications.
Night    →  Review. Adjust. Show up better tomorrow.
```

> *"Discipline is not about force. It's about clarity on why you're doing it."*

---

## ⚡ My Daily Operating System

| Time | Action |
|------|--------|
| 🌅 Morning | 10-min failure pre-mortem |
| 📋 Before work | Write the 3 non-negotiable tasks |
| 🚀 First 30 mins | Finish the easiest priority — build momentum |
| 🎧 Deep work | Lo-fi / instrumental only — no lyrics |
| 🌙 Before sleep | What worked? What didn't? What changes tomorrow? |

---

## 🎯 The Root Principle

Most people chase **willpower.**
I chase **clarity.**

When you know *exactly* why you're building something —  
the work stops feeling like work.
